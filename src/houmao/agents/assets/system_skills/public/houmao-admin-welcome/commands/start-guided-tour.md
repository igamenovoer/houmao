# Start Guided Tour

## Workflow

1. **Inspect only supported read-only posture** needed to distinguish blank, project-ready, running-agent, and loop-artifact states.
2. **Introduce Houmao in context** and use likely intent only to order choices.
3. **Present the state-appropriate menu**; use exactly Create Houmao Project, Subsystem Exploration, and Inspect for a blank workspace.
4. **Load one selected guided path** and teach its concepts, readiness, validation, and handoff points compactly.
5. **Offer non-linear continuation** or a context-preserving admin-entrypoint handoff without executing it.

If the request does not map cleanly to these steps, use the native planning tool to build a bounded read-only tour from orientation evidence, curated paths, and the user's goal, then begin at the nearest useful stage.
