# Show Options

## Workflow

1. **Keep comparison read-only** and use known posture only to order choices.
2. **Present the maintained guided paths** with their outcome, fit, and likely required context.
3. **Use the blank-workspace three-choice menu** when no project overlay exists.
4. **Ask one explanatory choice question** or offer `more detail` without executing a path.

If the request does not map cleanly to these steps, use the native planning tool to build a bounded read-only option set from the guided paths, current posture, and user goal, then present it.

Present the five curated paths from [../references/guided-paths.md](../references/guided-paths.md). Ask one explanatory choice question. Do not mutate state while comparing paths.
