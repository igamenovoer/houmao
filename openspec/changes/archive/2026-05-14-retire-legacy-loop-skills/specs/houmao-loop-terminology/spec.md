## ADDED Requirements

### Requirement: Retired loop package names are historical terminology
Houmao guidance SHALL treat retired package names for pairwise and generic loop skills as historical references rather than current routing targets.

Current generated loop material SHALL prefer pro topology terms: `tree-loop` and `generic-loop`.

#### Scenario: Current guidance names pro topology modes
- **WHEN** current Houmao loop guidance describes available loop topology choices
- **THEN** it presents `tree-loop` and `generic-loop` as pro topology modes
- **AND THEN** it does not present retired package names as current skill choices

#### Scenario: Historical pairwise wording is preserved only as legacy context
- **WHEN** documentation or validation discusses older artifacts that use pairwise wording
- **THEN** it may identify that wording as legacy tree-loop terminology
- **AND THEN** it does not route new work to retired pairwise package names
