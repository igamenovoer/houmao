# Enhancements

## Purpose
Track targeted improvement proposals, contract tightenings, and UX refinements
that are narrower than larger feature requests.

## Naming Pattern
`enhance-[description].md`
Example: `enhance-mail-send-requires-full-mailbox-address.md`

## Historical Snapshots
This directory may later include copied enhancement snapshots referenced by
archived OpenSpec or issue history. Keep stable paths when those references are
introduced.
