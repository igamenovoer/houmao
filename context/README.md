# Context

## Purpose
Centralized project knowledge base for AI-assisted development in `Houmao`.

## Structure
- `design/`: Technical design notes and architecture drafts.
- `hints/`: How-to guides and troubleshooting notes.
- `instructions/`: Reusable prompts, snippets, and command templates.
- `issues/`: Feature requests, known issues, and resolved issues.
- `logs/`: Session logs and implementation outcomes.
- `plans/`: Roadmaps and implementation plans.
- `refcode/`: External reference-only code and examples.
- `roles/`: Role-specific prompts and memory.
- `rules/`: Task-specific constraints and guardrails.
- `skills/`: Reusable skill packages/workflows.
- `summaries/`: Consolidated analyses and findings.
- `tasks/`: Work items grouped by status.
- `tools/`: Project-specific helper utilities.
