## TUI State Tracking Assets

The recordings under [`tui-state-tracking/`](./tui-state-tracking/) are intended as visual references for documentation, review, and implementation discussion around terminal state tracking.

Current recordings:

- `claude-state-tracking.mp4`
- `codex-state-tracking.mp4`

These videos are stored as H.264 (`libx264`) MP4 files and may be recompressed to keep repository size under control while preserving enough fidelity for terminal UI playback.
