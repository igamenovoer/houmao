# Legacy Demo Archive

Everything under this directory is archived historical material.

These packs are preserved for reference while the demo surface is redesigned. They are not supported runnable workflows, they are not covered by the active OpenSpec contract, and they should not block refactors of the live system.

Some archived packs still preserve historical references to the removed `tests/fixtures/agents/` fixture root. Those packs are expected to fail fast rather than to keep running against the maintained fixture lanes.
